La regresin lineal es un mtodo supervisado de machine learning que permite predecir el valor de una variable cuantitativa en base a otra variables o datos.

1. Introducción teórica a la regresión lineal simple¶

1.1. Ecuaciones y coeficientes¶

En estadstica la regresin lineal es un modelo matemtico usado para aproximar la relacin de dependencia entre una variable dependiente $y$, y la variable dependiente $x$. Este modelo asume que existe una relacin lineal entre la variable $x$ y la varible $y$, ms un trmino denominado error aleatorio que sigue una distribucin normal de media cero, $\epsilon$ ~ $N(0,\sigma)$. Matemticamente se puede expresar como:

\begin{align} y = f(x) + \epsilon = \alpha + \beta x + \epsilon \end{align}

Para saber si dos variables tienen una relacin de dependencia, es comn analizar la correlacin de stas. La correlacin $r$ de Pearson, es una medida de dependencia lineal entre dos variables aleatorias cuantitativas. A diferencia de otras medidas como la covarianza, la correlacin de Pearson es independiente de la escala de medida de las variables. El valor de este ndice oscila entre el intervalo [-1, 1], siendo los extremos donde se encuentra la correlacin mxima. La frmula es la siguiente:

\begin{equation*} r = \frac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum(x_i-\bar{x})^2\sum(y_i-\bar{y})^2}} \end{equation*}

Una vez se ha analizado la correlacin de dos variables, el siguiente paso es obtener el modelo de prediccin de de $y$ en base a $x$. En la primera ecuacin encontramos que existen dos parmetros desconocidos, $\alpha$ (intercepto u ordenada en el origen) y $\beta$ (pendiente), que ser necesario estimar. Una vez utilicemos nuestro data set de entrenamiento para obtener el valor de estos coeficientes, podremos predecir un valor $\hat{y}$ en base a la variable $x$.

\begin{equation*} \hat{y} = \hat{f}(x) = \hat{\alpha} + \hat{\beta} x \end{equation*}

Estos coeficientes se pueden obtener de forma analtica mediante el mtodo de los mnimos cuadrados, o de forma iterativa mediante el uso del algoritmo del gradiente descendente. En estos ejercicios, siempre se trabajar con el primero de ellos, el cual trata de minimizar la suma de los cuadrados de los residuos $e_i$ para todos y cada unos de los puntos del dataset, o dicho de otro modo, las diferencias entre el valor real de $y_i$ y el valor de la prediccin $\hat{y}_i$.

\begin{equation*} MSE = \frac{1}{n} \sum e_i^2 = \frac{1}{n} \sum (y_i - \hat{y}_i)^2 \end{equation*}

Las frmulas para estimar estos coeficientes derivadas del mtodo anterior son las siguientes:

\begin{equation*} \hat{\beta} = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2}\qquad \hat{\alpha} = \bar{y} - \hat{\beta} \bar{x} \end{equation*}

1.2. Precisión de los coeficientes¶

A la hora de realizar una regresin de este tipo y con el objeto de simplificar el modelo, se asume que la relacin entres las variables es puramente lineal, y se dejan fuera del modelo algunas otras variables como el trmino del error aleatorio $\epsilon$ u otras columnas del dataset. Por tanto, es claro que aunque nuestros coeficientes estimados $\hat{\alpha}$ y $\hat{\beta}$ sern similares los coeficientes reales $\alpha$ y $\beta$, no sern exactamente iguales.

Para obtener una estimacin del grado de precisin de los parmetros $\hat{\alpha}$ y $\hat{\beta}$ frente a los valores reales, se puede utilizar la frmula del error estndar (SE - Standard Error):

\begin{equation*} SE(\hat{\alpha})^2 = \sigma^2 \Bigl(\frac{1}{n} + \frac{\bar{x}^2}{\sum (x_i - \bar{x})^2} \Bigr) \end{equation*}\begin{equation*} SE(\hat{\beta})^2 = \frac{\sigma^2}{\sum (x_i - \bar{x})^2} \end{equation*}

Donde $\sigma^2$ es la varianza del error aleatorio $\epsilon$. De forma general este dato suele ser desconocido, pero se puede obtener una estimacin a partir de la frmula del error residual estndar:

\begin{equation*} RSE = \sigma = \sqrt{\frac{RSS}{n-2}} = \sqrt{\frac{\sum (y_i - \hat{y})^2}{n-2}} \end{equation*}

Estos errres estndar pueden ser utilizados para calcular intervalos de confianza. Un intervalo de confianza del 95% nos indica que existe una probabilidad del 95%, de que el coeficiente real desconocido est contenido en este rango. Para la regresin lineal, el intervalo de confianza del 95% para los dos coeficiente se obtiene mediante las siguientes frmulas:

\begin{equation*} \Bigl( \hat{\alpha} - 1.96SE(\hat{\alpha}),\ \hat{\alpha} + 1.96SE(\hat{\alpha}) \Bigr) \end{equation*}\begin{equation*} \Bigl( \hat{\beta} - 1.96SE(\hat{\beta}),\ \hat{\beta} + 1.96SE(\hat{\beta}) \Bigr) \end{equation*}

* En realidad, el coeficiente 1.96 de la frmula anterior tiende a este valor cuando el nmero de grados de libertad (n) aumenta y el porcentaje de inclusin en el intervalo es de un 95%. Si se quiere ser preciso en base a n, debera calcularse como el valor que contuviera el 97,5% de una distribucin t-student con n-2 grados de libertad.

Utilizamos el t estadstico ya que al trabajar con muestras de la poblacin total, no tenemos datos como la desviacin estndar.

Los errores estndar tambin pueden ser utilizados para llevar a cabo contrstes de hiptesis en los coeficientes. El contrste de hiptesis ms comn es contrastar si existe relacin entre X e Y.

  • Hiptesis nula $H_0$: No existe relacin entre X e Y
  • Hiptesis alternativa $H_\alpha$: Existe una relacin entre X e Y

Matemticamente:

\begin{equation*} H_0 : \beta_1 = 0\\ H_\alpha : \beta_1 \neq 0 \end{equation*}

Para ello calculamos un t-estadstico, que nos indica el nmero de desviaciones estndar que est $\hat{\beta}$ del valor 0.

\begin{equation*} t = \frac{\hat{\beta} - 0}{SE(\hat{\beta})} \end{equation*}

A partir de este valor, podemos obtener el p-valor, que es la probabilidad de que un nmero sea mayor o igual a $t$. Un p-valor pequeo nos indica una relacin entre X e Y, por lo que se podra rechazar la hiptesis nula.

Valores tpicos del p-valor para rechazar la hiptesis nula son los relativos al 1% y 5%. Cuando la n = 30, debemos fijarnos que el t-estadstica sea de 2 a 2,75.

1.3. Precisión del modelo¶

Una vez ha sido rechazada la hiptesis nula, es natural querer cuantificar como se adapta nuestro modelo a los datos. La calidad del ajuste de una regresin linea, se basa principalmente en dos valores, el $RSE$ y el $R^2$.

El $RSE$ es una estimacin de la desviacin estndar de $\epsilon$, o dicho de otro modo, es la cantidad media que se desviar la prediccin de la lnea de regresin verdadera. El $RSE$ es considerado una medida que indica la falta de adecuacin del modelo a los datos. Su ecuacin es:

\begin{equation*} RSE = \sigma = \sqrt{\frac{RSS}{n-2}} = \sqrt{\frac{\sum (y_i - \hat{y})^2}{n-2}} \end{equation*}

El $RSE$ es una medida de la discordancia del modelo a los datos, pero dado que es expresado en las unidades de la variables dependiente, no siempre es claro que valor de RSE es aceptable y cual no. El $R^2$ es una alternativa al indicador anterior, es el cuadrado del coeficiente de correlacion (Pearson). Este estadstico arroja un valor entre 0 y 1, y es independiente de la escala de Y. Para obtener este valor se utiliza la siguiente frmula:

\begin{equation*} R^2 = \frac{\sum(y_i - \bar{y})^2-\sum(y_i - \hat{y})^2}{\sum(y_i - \bar{y})^2} = \frac{TSS-RSS}{TSS} = 1 - \frac{RSS}{TSS} \end{equation*}

2. Introducción a la regresión lineal múltiple¶

2.1. Ecuaciones y coeficientes¶

Existen modelos en los que a veces no es suficiente el uso de una variable independiente $x$ para predecir una variable independiente $y$, es en estos casos donde entra en juego la regresin lineal mltiple, donde se tienen en cuenta un nmero $p$ de variables.

\begin{align} y = f(x) + \epsilon = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_p x_p + \epsilon \end{align}

Al igual que en el caso de la regresin lineal, estos coeficiente son desconocidos y tendrn que ser estimados para poder definir el modelo de prediccin de la variable $\hat{y}$.

\begin{align} \hat{y} = \hat{f}(x) = \hat{\beta}_0 + \hat{\beta}_1 x_1 + \hat{\beta}_2 x_2 + ... + \hat{\beta}_p x_p \end{align}

Para estimar estos coeficientes se utiliza el mtodo de los mnimos cuadrados, el cual trata de reducir para cada puntos el residuo, i.e., reducir la suma de los cuadrados de los residuos $RSS$. Dado que en la regresin lineal mltiple existen ms variables, ser necesario emplear mtodos matriciales para hallar analticamente los coeficientes.

\begin{align} Y = X\beta + \epsilon\qquad\rightarrow\qquad \begin{bmatrix}y_1\\y_2\\...\\y_n\end{bmatrix} = \begin{bmatrix}1&x_{11}&x_{12}&...&x_{1p}\\ 1&x_{21}&x_{22}&...&x_{2p}\\ ...&...&...&...&...\\ 1&x_{n1}&x_{n1}&...&x_{np}\end{bmatrix}\ \begin{bmatrix}\beta_o\\\beta_1\\\beta_2\\...\\\beta_p\end{bmatrix} + \begin{bmatrix}\epsilon_1\\\epsilon_2\\...\\\epsilon_n\end{bmatrix} \end{align}\begin{align} \beta = (X^TX)^{-1}X^TY \end{align}

Cuando tenemos datasets grandes, $np>10^6$, puede ser conveniente empezar a usar el algoritmo del gradiente descendiente, ya que este mtodo puede ralentizar bastante el script.

2.2. Precisón de los coeficientes¶

2.3. Precisión del modelo¶

\begin{equation*} RSE = \sigma = \sqrt{\frac{RSS}{n-p-1}} = \sqrt{\frac{\sum (y_i - \hat{y})^2}{n-p-1}} \end{equation*}

6. Regresión lineal con variables categóricas¶

Hasta ahora, en todos los casos desarrollados, las variables predictoras eran numricas o cuantitativas, pero se pueden encontrar datasets que tengan variables categricas con un impacto importante en la variable a predecir.

Para solucionar este problema, lo que se suele hacer es crear variables dummies por cada variable categrica que se quiera incluir en el modelo predictor, e incluirlas en las ecuaciones de regresin.
